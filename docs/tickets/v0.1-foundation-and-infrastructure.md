## Scope

### Backend (AdonisJS 6)

- Scaffold AdonisJS 6 project inside `backend/` using:
  ```bash
  npm create adonisjs@latest backend -- --kit=api --db=postgres --auth-guard=access_tokens
  ```
- Configure PostgreSQL driver (`@adonisjs/lucid`) and [Valkey](https://valkey.io/) connection via `@adonisjs/redis` (Valkey is Redis-compatible; config in `config/redis.ts`)
- Encryption is provided by `@adonisjs/core`'s built-in encryption service, keyed by `appKey` in `config/app.ts`. No separate `config/encryption.ts` is needed.
- Write and run migrations:
  - `users` â€” `id` (UUID PK), `first_name`, `last_name` (enc), `email` (enc), `password_hash`, `birth_date` (enc), `profile_type` (`teacher` | `student`), `avatar_url`, `created_at`, `updated_at`, `deleted_at` (soft delete)
  - `access_tokens` â€” generated by `@adonisjs/auth` access-token guard
  - `rate_limits` â€” generated by `@adonisjs/limiter` (required when `LIMITER_STORE=database`)
  - `student_profiles` â€” `id`, `user_id` (FK), `weight_kg` (enc, nullable), `height_cm` (enc, nullable), `data_consent_at`
  - `teacher_profiles` â€” `id`, `user_id` (FK), `fight_experience` (nullable)
  - `audit_logs` â€” `id`, `user_id`, `action`, `resource_type`, `resource_id`, `ip_address`, `metadata` (JSON), `created_at`
- Set up `@adonisjs/cors` (strict: only `VITE_APP_URL` origin allowed, no wildcard)
- Set up `@adonisjs/limiter` (wired, not yet applied to routes)
- Global exception handler that returns consistent JSON errors and never leaks stack traces, DB details, or internal paths
